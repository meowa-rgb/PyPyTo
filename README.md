# 📦 PyPyTo v1.0

**PyPyTo** — это ультра-простой графический компилятор Python скриптов в независимые `.exe` файлы для Windows. Больше не нужно страдать с консолью и путями PyInstaller!

## ✨ Что умеет программа:
* 🚀 Компиляция `.py` в один клик.
* 🖼️ Автоматическая конвертация ЛЮБЫХ картинок (PNG, JPG, JPEG) в формат `.ico` для иконки твоей программы.
* 👁️ Возможность скрыть консольное окно (для GUI-приложений) одной галочкой.
* 📂 Автоматическая изоляция сборки в отдельную чистую папку, чтобы не засорять рабочий стол.

## 📥 Как скачать:
Перейдите в раздел **Releases** (справа на этой странице), скачайте архив `PyPyTo_v1.0.zip`, распакуйте в любое место и запускайте!

> ⚠️ **Важно:** Для работы программы на компьютере должен быть установлен обычный Python. При первом запуске Windows SmartScreen может выдать предупреждение — это нормальное ложное срабатывание для всех собранных через PyInstaller программ.
