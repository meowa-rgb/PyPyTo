# PyPyTo
**PyPyTo v1.0** — удобный GUI-компилятор Python в `.exe`. Сделал для себя, чтобы не страдать с консолью и PyInstaller. В один клик собирает скрипты, на лету конвертирует любые картинки (PNG, JPG) в `.ico` для иконки и умеет скрывать окно консоли. Все файлы сборки аккуратно изолируются в отдельную папку. Пользуйтесь на здоровье! Мяу! 📦🐾
